flutter 安装

#
在windows参考下面设置两个环境变量
在linux下输入下面命令
```
export PUB_HOSTED_URL=https://pub.flutter-io.cn
export FLUTTER_STORAGE_BASE_URL=https://storage.flutter-io.cn
```
下载flutter后解压执行 flutter doctor 等待更新完成
```commandline
 flutter doctor
```
